\section{Data and Transformations}
Raw data are read from \texttt{data/raw/}. Processed model-ready data are written to
\texttt{data/processed/}. The HSA competition aggregator convention is
$N_t = 100\log(N^{level}_t)$ unless a run metadata file explicitly states
that the series was already transformed.

\section{Model Specifications}
The CES benchmark is
\[
\pi_t=\alpha \pi_{t-1}+(1-\alpha)E_t\pi_{t+1}+\kappa x_t+e_t.
\]
The dynamic HSA specification is
\[
\pi_t=\alpha \pi_{t-1}+(1-\alpha)E_t\pi_{t+1}+\kappa x_t-\theta\hat N_t+e_t.
\]
The steady HSA specification is
\[
\pi_t=\alpha \pi_{t-1}+(1-\alpha)E_t\pi_{t+1}+(\kappa_0+\delta\bar N_t)x_t+e_t.
\]
The both-effects HSA specification is
\[
\pi_t=\alpha \pi_{t-1}+(1-\alpha)E_t\pi_{t+1}
+(\kappa_0+\delta\bar N_t)x_t-(\theta_0+\gamma\bar N_t)\hat N_t+e_t.
\]
The maintained HSA decomposition is
\[
N_t=\hat N_t+\bar N_t,\quad
\hat N_t=\rho_1\hat N_{t-1}+\rho_2\hat N_{t-2}+u_t,\quad
\bar N_t=n+\bar N_{t-1}+\epsilon_t.
\]

\section{Unit and Covariance Conventions}
Priors for \(\kappa\), \(\kappa_0\), and \(\delta\) are specified in
physical units. Legacy Gibbs samplers internally use \texttt{KAPPA\_SCALE=100}
when the design matrix contains \(x_t/100\). All posterior draws, tables,
figures, SDDR calculations, and model-comparison summaries are reported in
physical units. The default dynamic-HSA shock covariance allows only
\((e_t,\zeta_t)\) correlation; diagonal and historical full covariance modes
are available only through explicit run metadata.

\section{Prior Specifications}
Baseline, weak, and tight priors are stored in \texttt{configs/}. Robustness
tables below summarize whether conclusions for \(\delta\), \(\theta\), and
\(\gamma\) change across these choices.
